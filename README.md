# ⏱️ 타임싱크 (TimeSync)
> **대학생을 위한 똑똑한 올인원(All-in-One) 공강 및 일정 관리 시스템**

---

## 💡 무엇을 하는 프로젝트인가

**타임싱크**는 복잡한 대학 생활 속에서 **수업, 아르바이트, 개인 약속**을 한눈에 관리하고, 자동으로 **비어있는 공강 시간(교시)**을 찾아주는 모바일 최적화 웹 앱입니다. 

매 학기 바뀌는 시간표와 불규칙한 알바 시간 때문에 친구들과 팀플이나 밥 약속 잡기 힘들었던 경험에서 출발했습니다. 캘린더와 시간표를 번번이 대조할 필요 없이, 입력된 데이터를 기반으로 요일별 공강을 직관적으로 계산해 주어 대학생들의 효율적인 시간 관리를 돕습니다.

---

## ✨ 주요 기능

* **📊 홈 대시보드:** 주간 총 수업 교시, 알바 횟수, 다가오는 약속, 오늘 비어있는 공강 교시를 한눈에 요약해서 보여줍니다.
* **📚 인터랙티브 시간표:** 요일과 교시를 선택해 내 시간표를 채우고, 등록된 수업과 알바 일정이 시간표 내에 다른 색상으로 직관적으로 표시됩니다.
* **💼 알바 일정 관리:** 불규칙한 알바 요일과 시작/종료 시간을 등록하여 고정 시간표와 겹치지 않게 관리할 수 있습니다.
* **📅 스마트 캘린더:** 날짜별로 등록된 수업, 알바, 개인 약속을 도트(Dot) 배지로 구분하여 시각화하고, 상세 일정을 하단에 띄워줍니다.
* **🕊 평일 빈 교시 찾기:** 수업과 알바가 없는 청정 공강 시간(1~10교시)을 요일별로 자동 계산해 주어 공강 활용 계획이나 약속 조율을 쉽게 만들어줍니다.

---

## 🛠 기술 스택

* **Frontend:** HTML5, CSS3 (Modern UI / CSS Variables), JavaScript (Vanilla JS)
* **Storage:** 브라우저 내장 `localStorage`를 활용한 실데이터 저장 및 관리 (서버 없이도 데이터 유지)
* **Design:** 토스/당근마켓 스타일의 깔끔한 모바일 퍼스트(Mobile-First) 디자인
